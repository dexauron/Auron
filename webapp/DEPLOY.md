# Развёртывание веб-приложения

## 1. Создать Google Таблицу

1. Перейдите на [sheets.google.com](https://sheets.google.com) → создайте новую таблицу
2. Скопируйте **ID** из URL (часть между `/d/` и `/edit`):
   `https://docs.google.com/spreadsheets/d/**ВСТАВЬТЕ_ЭТОТ_ID**/edit`

## 2. Открыть Apps Script

В таблице: **Расширения → Apps Script**

## 3. Загрузить файлы

### Code.gs
Замените содержимое `Code.gs` на содержимое файла `webapp/Code.gs`.

В строке `var SS_ID = '';` вставьте ID таблицы:
```javascript
var SS_ID = 'ВАШ_ID_ЗДЕСЬ';
```

### Index.html
Создайте новый файл HTML: **+ → HTML** → назовите `Index` (без расширения).
Вставьте содержимое `webapp/Index.html`.

## 4. Развернуть

1. Нажмите **Развернуть → Новое развёртывание**
2. Тип: **Веб-приложение**
3. Выполнять как: **Я (ваш аккаунт)**
4. Доступ: **Все** (или «Только я» для приватного использования)
5. Нажмите **Развернуть** → скопируйте ссылку

## 5. Установить на iPhone как PWA

1. Откройте ссылку в Safari на iPhone
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой вверх)
3. Выберите «На экран «Домой»»
4. Название: «Финконтроль» → Добавить

Приложение появится на главном экране как нативное iOS-приложение.

## Структура листов (создаются автоматически)

| Лист | Описание |
|------|----------|
| БАЗА | Все операции (дата, смена, кассир, тип, категория, способ, сумма, расхождение) |
| НАСТРОЙКИ | Справочники: кассиры, смены, категории, способы оплаты, поставщики |

Листы создаются автоматически при первом запуске.
Вы можете заполнить НАСТРОЙКИ своими данными.
