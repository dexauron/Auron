# 07 — Банк идей

> Одобренные владельцем идеи, ещё не вошедшие в основные документы.
> При проектировании каждой версии — выбирать отсюда. Идея реализована → переносится в 02/03 и вычёркивается здесь.

## Скорость и ввод
| Идея | Суть |
|------|------|
| Темп дня | «Сейчас 14:00, обычно к этому времени ₽38К, сегодня ₽51К 🟢 +34%» |
| Утренний брифинг | Карточка «Ожидаем сегодня»: кто придёт, сколько нужно, хватает ли |
| Шаблоны из паттернов | «Каждый понедельник Иванов ~₽35К» → предлагает запланировать сам |
| Голосовой ввод с Главной | Зажал микрофон → «Иванов заплатил 35 тысяч» → форма заполнена |
| «Как обычно» | Кнопка в нумпаде если сумма+поставщик совпадают с историей |
| Похожие транзакции | Ввёл ₽35 000 → «Обычно это Иванов П. — добавить?» |
| Умный буфер обмена | Скопировал число → открыл нумпад → сумма вставилась |
| Drag & drop в Календаре | Перетащил выплату на другой день → всё пересчиталось |
| Массовые действия | Выделить несколько транзакций → изменить категорию / удалить / экспорт |
| Сканер штрих-кода | Ручной учёт товаров: навёл камеру → название из открытой базы |

## Умные подсказки и ИИ
| Идея | Суть |
|------|------|
| Аномалия при вводе | Z-отчёт вдвое ниже среднего → «Всё верно?» до сохранения |
| Детектив расхождений | ИИ находит паттерн: «73% расхождений — ночные смены по пятницам» |
| «Умный день» | Утренний план от ИИ: с чего начать, что не забыть |
| Прогноз выручки на неделю | По дням с диапазоном погрешности и уровнем уверенности |
| Дельта-отчёт | «Что изменилось за сутки» — только изменения, не полный отчёт |
| Финансовый автопилот | Сам считает сколько отложить в накопления из выручки дня → [Подтвердить] |
| Инсайты при скролле | Листаешь старый месяц → тихий контекст «год назад было на ₽18К выше» |

## Поставщики
| Идея | Суть |
|------|------|
| История цен | График среднего чека накладных: «+25% за полгода» |
| Сравнение поставщиков | Два поставщика бок о бок: суммы, точность, пунктуальность |
| «Поставщик пропал» | Не появлялся дольше обычного → предложение написать в WhatsApp |
| Проверка контрагента | Кнопка → запрос в ЕГРЮЛ/ЕГРИП → статус компании |
| QR-код поставщика | Наклейка на стеллаже → скан → карточка с «Записать накладную» |

## Команда
| Идея | Суть |
|------|------|
| «Отчёт прочитан» | Бухгалтер видит: «👁 Владелец прочитал в 20:14» |

## Платформа и устройства
| Идея | Суть |
|------|------|
| Виджет на рабочий стол | Пульс + выручка + касса без открытия приложения |
| Шорткаты иконки | Долгое нажатие → Z-отчёт / Выплата / Транзакция |
| «Поделиться карточкой» | Любой график → красивый скриншот → WhatsApp |
| Хаптика | Успех/ошибка/предупреждение — разные вибрации |
| Часы (watch) | Три цифры: выручка · касса · пульс |
| NFC-метка на кассе | Приложил телефон → открылся Z-отчёт |
| Карта сети | Магазины на карте города, цвет точки = пульс |
| Режим владельца | Отдельный простой экран: 4 цифры и одна кнопка |
| Голосовые заметки | К любой записи — голосовое 10–30 сек |
| Фото к записи | Не только накладные — любые документы |

## Отложено / требует нативного приложения
| Идея | Причина |
|------|---------|
| SMS-парсинг банковских уведомлений | PWA не имеет доступа к SMS — ждёт нативного Android-приложения |
| Push-кнопки действий на iOS | Веб-push на iPhone не поддерживает кнопки |
| Shake to Undo | Требует разрешения на датчики в iOS — проверить на практике |

## Отложено до v2+
| Идея | Причина |
|------|---------|
| Мультивалютность (USD, EUR, другие валюты) | v1 только рубли; 99% Way Market-пользователей работают в рублях; правильная реализация требует курсов обмена и пересчёта исторических данных |
