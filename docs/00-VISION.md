# 00 — Видение продукта

> Статус: ✅ утверждён (обновлён — расширение до универсальной финансовой платформы)

## Auron Финансы — это…

Универсальная финансовая платформа для любого кто управляет деньгами. Работает для частного человека как личный трекер финансов — и для бизнеса любого типа как операционная система учёта. Один продукт, бесконечное число конфигураций.

**Ближайший аналог по духу:** Zenmoney — но не только для личных финансов, а для любого масштаба и любой сферы.

## Для кого

| Кто | Как использует |
|-----|---------------|
| Частный человек | Личный бюджет, расходы, цели, накопления |
| ИП / фрилансер | Доходы от клиентов, расходы по проектам, налоги |
| Малый бизнес | Касса, поставщики, сотрудники, аналитика |
| Розничный магазин | Z-отчёты, долги поставщикам, табель, рентабельность |
| Кафе / ресторан | Смены, закупки, себестоимость блюд |
| Салон красоты | Мастера, услуги, клиенты, расходные материалы |
| Сеть магазинов | Несколько точек из одного аккаунта, сводная аналитика |

**Первый клиент и первый модуль:** Way Market — розничный магазин. Это v1.0. На его примере строится эталонная реализация «Модуля розницы».

## Архитектура продукта

**Ядро (универсальное для всех):**
- Счета (наличные, карта, банк, накопления — любые)
- Транзакции (доход / расход / перевод)
- Категории (настраиваются под себя)
- Бюджеты и накопления
- Цели
- Аналитика и отчёты

**Модули (включаются под тип бизнеса или задачу):**
- Модуль «Розница» — Z-отчёты, поставщики с долгами, табель сотрудников
- Модуль «Услуги» — запись клиентов, мастера, прайс
- Модуль «Проекты» — доходы/расходы по проектам (для фрилансеров и ИП)
- Модуль «ИИ» — прогнозы, аномалии, вопросы к данным
- Модуль «Команда» — несколько пользователей, роли, доступы
- … и новые модули добавляются без изменения ядра

## Бизнес-модель: Freemium SaaS

### Бесплатно — лучше чем Excel и блокнот
- Все базовые операции ядра (счета, транзакции, категории, бюджеты)
- Один активный модуль на выбор
- До 1 организации / профиля
- Базовая аналитика и отчёты

### Платно — то чего Excel и блокнот никогда не дадут
- Все модули без ограничений
- ИИ-ассистент (вопросы к данным, прогнозы, аномалии)
- Несколько организаций / магазинов
- Команда (несколько пользователей с ролями)
- Автоматические уведомления и WhatsApp-отчёты
- Журнал изменений (аудит)
- Расширенная аналитика и экспорт

## Чего продукт НЕ делает — принципиально
- ❌ Не заменяет 1С (складской учёт, ТСД, ценники)
- ❌ Не принимает платежи (не POS-система)
- ❌ Не ведёт CRM (управление клиентской базой — отдельный продукт)
- ❌ Не является бухгалтерским ПО для сдачи отчётности в налоговую

## Критерий успеха
Любой человек — хоть частник который хочет понять куда уходят деньги, хоть владелец сети магазинов — открывает Auron Финансы и за 5 минут понимает своё финансовое состояние без звонков и таблиц.
